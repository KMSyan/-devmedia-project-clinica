Checklist - html

Para garantir que seu passo 1 está completo, verifique os itens abaixo:

- Certifique-se de que a hierarquia dos elementos HTML está correta
- Certifique-se ter inserido o título da página (tag Title)
- Certifique-se ter definido o charset da página (meta tag charset UTF-8)
- Certifique-se de que a estrutura HTML usa tags do HTML semântico;
- Certifique-se de que ao clicar na logo do topo a página index.html seja
 carregada;
- Certifique-se de que os links âncora do topo da página estão apontando para
 os locais corretos. São eles:
  Sobre
  Contato
- Certifique-se de que as imagens estão sendo exibidas na tela;
- Certifique-se de que o iframe está sendo exibido na tela.

Checklist - css

Para garantir que seu passo 2 está completo, verifique os itens abaixo:

- Certifique-se de importar e usar a fonte 'Source Sans Pro'
- Certifique-se de ter inserido as classes CSS nas tags HTML (arquivo index.html).
- Certifique-se de que os links com texto do topo mudem de cor quando o mouse 
estiver sobre eles.
- Certifique-se de que a imagem de fundo da primeira divisão do conteúdo esteja aparecendo.
- Certifique-se de que as demais imagens estejam aparecendo.
- Certifique-se de que as classes estão sendo reutilizadas quando possível (você não 
deve criar diferentes classes que possuem o mesmo estilo CSS).
- Certifique-se de que os containers que limitam a largura dos seus conteúdos a 1024px 
fiquem centralizados na página.

- Certifique-se de que a meta tag viewport foi inserida no HTML da página home (arquivo

Checklist

Para garantir que seu passo 4 está completo, verifique os itens abaixo:

- Certifique-se ter inserido o título da página (tag Title) e definido o charset da página 
(meta tag charset UTF-8) dentro da tag <head></head>.
- Certifique-se de que a estrutura HTML usa tags do HTML semântico.
- Certifique-se de que os links âncora estão apontando para os locais corretos. São eles:
  - Ortopedia
  - Pediatria
  - Geriatria
  - Psicologia
  - Fisioterapia
- Certifique-se de que as imagens estão sendo exibidas na tela.
- Certifique-se de que os links do topo apontam para a página correta:
   - A logo deve abrir a página home.
   - O link "Sobre" deve abrir a página home na sessão "Sobre a clínica".
   - O link "Especialidades" deve abrir a própria página Especialidades.
   - O link "Contato" deve abrir a página home na sessão que possui os dados de contato e o iframe.
   - Na página home certifique-se de que o link "Especialidades" no topo e o link "Ver horário" 
 estejam apontando para a página Especialidades.
